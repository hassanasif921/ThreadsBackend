API Testing - Complete User Flow

Authentication Status

This will be automatically populated when you login below

User Registration

Endpoint: POST /api/users/register

User Login

Endpoint: POST /api/users/login

Email Verification

Endpoint: POST /api/users/verify-email

Note: For testing, use OTP "0000"

Profile Update API Test

Endpoint: POST /api/users/profile

Authentication: Required (JWT Token)

Step Update API Test

Endpoint: PATCH /api/stitches/:stitchId/steps/:stepId

Authentication: Required (JWT Token)

Complete User Flow

  1. Register a new user
  2. Verify email with OTP "0000"
  3. Login to get JWT token
  4. Update profile (optional)
  5. Test step updates (need valid IDs)
